Cyber Bounce 3000

Cyber Bounce 3000 is a multiplayer VR game where guests compete by throwing balls to each other and try to dodge or defend by shield.

Gameplay Video



Project Infomation


Duration 3 weeks (Oct. 31 - Nov. 20, 2018)
Team Size 5 (2 Programmers + 2 Artists + 1 Sound Designer)
Theme Make an interactive experience for festival
Platform 2 HTC Vives
Team members Shengzhi Wu, Yanyu (Wren) Tong, Linyao (Freya) Li, Zhaoyi (Ashley) Liang

My Contributions


Programming:

  1. Learned Unity networking tool "Unet" and implemented the structure of a Server-Client system to hold a low latencyt gameplay.
  2. Implemented gameplay of throwing balls and bouncing, as well as the system for player preparation.
  3. Assisted artists on making environment and setting up models in Unity.

Design:

  1. Took part in designing game mechnism where two guests compete by throwing balls to hit each other.
  2. Helped on designing playtests and adjusting parameters such as speed of the ball, length and height of the pipeline.

Lesson


This is the first time I worked on games using network functions so the largest gain would be grasping the usage of Unet. For this, I read the Unity tutorials as well as the code on Bitbucket. In addition, I found how difficult it is to simulate throwing ball in VR. We played games like Sparc to learn about the feeling of throwing balls, but it also feels not that good as real. Last but not least, I gradually learned the importance of playtesting and how to make the game better through tests.
